Security Bulletin

A Microsoft számú biztonsági közleménye MS08-077 - Fontos
=========================================================

A Microsoft Office SharePoint Server biztonsági rései jogok kiterjesztését okozhatják (957175)
----------------------------------------------------------------------------------------------

Közzétéve: 2008. december 9.

**Verzió:** 1.0

### Általános tudnivalók

#### Összefoglalás

Ez a biztonsági frissítés egy közvetlenül jelzett biztonsági rést szüntet meg. A biztonsági rés lehetővé teheti a jogok kiterjesztését, ha a támadó kikerüli a hitelesítést egy felügyeleti URL-címre vagy SharePoint-webhelyre navigálva a böngészővel. A jogok kiterjesztéséhez vezető sikeres támadás következménye szolgáltatásmegtagadás vagy adatokhoz való illetéktelen hozzáférés lehet.

Ez a fontos besorolású biztonsági frissítés a Microsoft Office SharePoint Server 2007 és a Microsoft Search Server 2008 összes támogatott kiadására vonatkozik. További információkért lásd az **Érintett és nem érintett szoftverek** részt ebben a fejezetben.

A biztonsági réssel kapcsolatban bővebben az adott problémával foglakozó gyakran feltett kérdéseket ismertető szakaszban, **A biztonsági réssel kapcsolatos tudnivalók** című részben olvashat.

**Javaslat.** A Microsoft a biztonsági frissítés lehető leghamarabb történő telepítését javasolja ügyfeleinek.

**Ismert problémák** A [Microsoft Tudásbázis 957175 számú](http://support.microsoft.com/kb/957175) cikkében olvasható, hogy a felhasználók a biztonsági frissítés telepítésekor milyen ismert problémákkal találkozhatnak. A cikkben a problémákra javasolt megoldások is szerepelnek.

#### Érintett és nem érintett szoftverek

Az itt felsorolt szoftvereket a verziók és változatok érintettségének felderítése érdekében teszteltük. A többi verzió, illetve kiadás támogatási életciklusa végére ért vagy nem érintett. Az egyes szoftverek és azok verzióinak támogatási életciklusáról a [Microsoft termékek terméktámogatási életciklusát ismertető webhelyen](http://go.microsoft.com/fwlink/?linkid=21742) talál felvilágosítást.

Hozzáférés-vezérlés biztonsági rése - CVE-2008-4032  
---------------------------------------------------
  
<span></span>
A Microsoft Office SharePoint Server 2007 és Microsoft Office SharePoint Server 2007 Service Pack 1 biztonsági rése jogok megszerzéséhez vezet. A biztonsági rés lehetővé teheti a jogok kiterjesztését, ha a támadó kikerüli a hitelesítést egy felügyeleti URL-címre vagy SharePoint-webhelyre navigálva a böngészővel. A jogok kiterjesztéséhez vezető sikeres támadás következménye szolgáltatásmegtagadás vagy adatokhoz való illetéktelen hozzáférés lehet.
  
A biztonsági rés standard bejegyzésének megtekintéséhez a normál biztonsági rés besorolási listáján lásd: [CVE-2008-4032](http://www.cve.mitre.org/cgi-bin/cvename.cgi?name=cve-2008-4032).
  
#### A hozzáférés-vezérlés biztonsági résének (CVE-2008-4032) súlyosságát enyhítő tényezők
  
A Microsoft egyelőre nem talált enyhítő tényezőket a biztonsági rés megszüntetésére.
  
#### Lehetséges megoldások a hozzáférés-vezérlés biztonsági résének (CVE-2008-4032) megszüntetésére:
  
A lehetséges megoldás olyan beállítást vagy konfigurációs módosítást takar, amely magát a biztonsági rést nem szünteti meg, azonban a frissítés telepítése előtt segít az ismert támadási lehetőségek kivédésében. A Microsoft alaposan megvizsgálta a következő lehetséges megoldásokat, és tájékoztatja a felhasználókat, amennyiben a megoldás csökkenti a működőképességet:
  
-   **Rosszindulatú HTTP-kérelmek blokkolása tűzfal használatával**
  
    Konfigurálja a tűzfalat (például Microsoft ISA Server) az összes HTTP-kérelem blokkolására, amelyek az URL-címben az „mode=ssp” elemet tartalmazó, de a „/ssp/admin/\_layouts” elemet nem tartalmazó SharePoint-webhelyekre vonatkoznak.
  
#### Gyakran ismételt kérdések a hozzáférés-vezérlés biztonsági résével (CVE-2008-4032) kapcsolatban
  
**Mire terjed ki a biztonsági rés hatása?**    
A Microsoft Office SharePoint Server 2007 és Microsoft Office SharePoint Server 2007 Service Pack 1 biztonsági rése jogok megszerzéséhez vezet, mely által egy hitelesítés nélküli felhasználó a SharePoint-kiszolgáló azon oldalára navigálhat, amely felügyeleti szolgáltatások végrehajtására szolgál. A biztonsági rést sikeresen kihasználó támadó szolgáltatásmegtagadást idézhet elő a kiszolgálót túlterhelő parancsok végrehajtásával. A támadó, továbbá, bizalmas adatokhoz juthat a Microsoft Office SharePoint Services szolgáltatásoktól (például belső elérési útvonalak és felhasználók e-mail címei), és a webhely környezetében futó parancsfájlokat hozhat létre.
  
**Mi okozza a biztonsági rést?**    
A Microsoft Office SharePoint Server 2007 alkalmazás hibásan kezeli bizonyos felügyeleti szolgáltatások hozzáférés-vezérlését.
  
**Mire használhatja a biztonsági rést a támadó?**    
A biztonsági rést sikeresen kihasználó támadó kiterjesztett jogokra tehet szert az érintett rendszeren, és ezáltal szolgáltatásmegtagadást idézhet elő a kiszolgálót túlterhelő parancsok végrehajtásával. A támadó, továbbá, a Microsoft Office SharePoint Server 2007 alkalmazással kapcsolatos bizalmas adatokhoz juthat, mint például belső elérési útvonalak és felhasználók e-mail címei.
  
**Hogyan használhatja ki a támadó a biztonsági rést?**    
A támadó ezt a biztonsági rést úgy próbálhatja meg kihasználni, hogy webböngészővel a SharePoint-kiszolgáló felügyeleti szolgáltatások végrehajtására szolgáló sebezhető oldalára navigál, és ezáltal kerüli ki a hitelesítést.
  
**Mely rendszerekre jelenti a legnagyobb veszélyt a biztonsági rés?**    
A Microsoft Office SharePoint Server 2007 és Microsoft Office SharePoint Server 2007 Service Pack 1 alkalmazásokat futtató kiszolgálók, és az ezekhez kapcsolódó munkaállomások vannak elsősorban kitéve a jogok kiterjesztéséből adódó támadások kockázatának. A sebezhető SharePoint Server alkalmazást futtató kiszolgálókon a jogok kiterjesztése szolgáltatásmegtagadáshoz vezethet. A sebezhető SharePoint-webhely hozzáféréséhez használt munkaállomások esetében a jogok kiterjesztése bizalmas adatokhoz való illetéktelen hozzáféréssel járhat.
  
**Mi a frissítés feladata?**    
A frissítés módosítja a Microsoft Office SharePoint Server 2007 és Microsoft Office SharePoint Server 2007 alkalmazások hitelesítési eljárását az érintett felügyeleti szolgáltatások esetében.
  
**Ismert volt-e a nyilvánosság előtt ez a biztonsági rés a biztonsági közlemény kiadásának idején?**    
Nem. A Microsoft ezt a célt szolgáló tájékoztatáson keresztül szerzett tudomást a biztonsági résről. A Microsoft nem kapott olyan információt, hogy e közlemény eredeti változatának megjelenéséig a biztonsági rést a nyilvánosság tudomására hozták volna.
  
**A közlemény kibocsátásáig kapott-e a Microsoft a biztonsági rés kihasználásáról tájékoztató értesítést?**    
Nem. E közlemény eredeti verziójának kiadásáig a Microsoft nem kapott olyan információt, amely arra utalt volna, hogy ezt a biztonsági rést nyilvános támadásra használták, és nem találkozott olyan nyilvánosságra hozott leírásokkal, amelyek tartalmazták volna egy sikeres támadás forgatókönyvét.
